# 💼 Portfólio Pessoal – Jones Mendes

Este projeto é um **site de portfólio profissional**, criado com o objetivo de apresentar meus projetos, habilidades e trajetória como desenvolvedor front-end, servindo como um cartão de visitas digital para recrutadores, clientes e parceiros.

## 🎯 Objetivo do Projeto

O portfólio foi desenvolvido para:

- Centralizar meus principais projetos em um único lugar
- Demonstrar minhas habilidades técnicas em desenvolvimento front-end
- Apresentar minha identidade profissional
- Facilitar o contato para oportunidades de trabalho, freelas ou parcerias
- Servir como vitrine prática do meu conhecimento em tecnologias web modernas

---

## 🚀 Tecnologias Utilizadas

### 🌐 Front-end
- Estrutura moderna de desenvolvimento web
